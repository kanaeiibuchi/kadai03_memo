メモパッドアプリのリポジトリ
（１）課題番号-プロダクト名
自己紹介カード

（２）課題内容（どんな作品か）
自己紹介のカードを共有できます
フォームに項目を入力するとカードが生成されてカードを並べてみることができます

（３）工夫した点・こだわった点
全然だけど自分的にはようやくCSSに手を入れる余裕ができた

（４）難しかった点・次回トライしたいこと(又は機能) 
CSSで謎挙動が出てきてチューターさんに協力してもらって解決したけど
絶対に自分では見つけられなかった。
